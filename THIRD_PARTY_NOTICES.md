# Third-Party Notices

This plugin includes or integrates with the following third-party services and libraries.

---

## Google reCAPTCHA v3

- **Website:** https://www.google.com/recaptcha/
- **Terms of Service:** https://policies.google.com/terms
- **Privacy Policy:** https://policies.google.com/privacy
- **Usage:** Optionally loaded on the frontend when reCAPTCHA v3 is enabled for a form. The reCAPTCHA API script is fetched from `https://www.google.com/recaptcha/api.js`. Enabling this feature means user data (including IP address) is transmitted to Google.

---

## Cloudflare Turnstile

- **Website:** https://www.cloudflare.com/products/turnstile/
- **Terms of Service:** https://www.cloudflare.com/website-terms/
- **Privacy Policy:** https://www.cloudflare.com/privacypolicy/
- **Usage:** Optionally loaded on the frontend when Cloudflare Turnstile is enabled for a form. The Turnstile widget script is fetched from `https://challenges.cloudflare.com/turnstile/v0/api.js`. Enabling this feature means user data (including IP address) is transmitted to Cloudflare.

---

No third-party PHP libraries are bundled in this plugin.
