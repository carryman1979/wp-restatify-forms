# Restatify Forms 1.0.2

## Was ist neu

- Verhalten im Formular-Editor korrigiert: Das Platzhalterfeld wird fuer `select`-Felder (Dropdown) nicht mehr angezeigt.

## Warum

Dropdown-Felder nutzen vordefinierte Optionen. Ein freies Platzhalter-Feld im Editor war daher irrefuehrend und praktisch ohne Nutzen.

## Kompatibilitaet

- Plugin-Version: `1.0.2`
- Keine Datenmigration erforderlich
