# Restatify Forms 1.0.3

## Was ist neu

- Der gemeinsame WYSIWYG-Workflow fuer Mail-Templates wird nun auch in Forms wiederverwendet.
- HTML-Mailtexte fuer Inhaber- und Bestaetigungs-Mail werden jetzt im TinyMCE/Quicktags-Editor bearbeitet.
- Platzhalter-Chips werden direkt in den aktiven Rich-Text-Editor eingefuegt.

## Warum

Nach dem Refactor wurden Mail-Templates in Forms noch als einfache Textareas bearbeitet. Dieses Release gleicht Forms an das gleiche Rich-Editing-Verhalten wie im Booking-Plugin an.

## Kompatibilitaet

- Plugin-Version: `1.0.3`
- Keine Datenmigration erforderlich
