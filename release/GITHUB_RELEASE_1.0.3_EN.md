# Restatify Forms 1.0.3

## What's new

- Reused the shared WYSIWYG mail-template editing workflow for Forms.
- Owner and confirmation HTML mail bodies now open in TinyMCE/Quicktags editor mode.
- Placeholder chips now insert directly into the active rich-text editor.

## Why

Forms still edited mail templates as plain textareas after the refactor. This release aligns Forms with the same rich editing experience used in Booking.

## Compatibility

- Plugin version: `1.0.3`
- No data migration required

- `wp-restatify-forms-1.0.3.zip`
