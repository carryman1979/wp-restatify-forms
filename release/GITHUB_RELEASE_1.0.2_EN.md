# Restatify Forms 1.0.2

## What's new

- Fixed form editor behavior: placeholder input is no longer shown for `select` (dropdown) fields.

## Why

Dropdown fields use predefined options, so a free-text placeholder setting in the editor caused confusion and had no practical value.

## Compatibility

- Plugin version: `1.0.2`
- No data migration required

## Artifact

- `wp-restatify-forms-1.0.2.zip`
