# Schnellstart

## Installation

1. Plugin ZIP unter Plugins -> Installieren -> Plugin hochladen installieren.
2. Plugin aktivieren.
3. Im Admin-Menue RS Forms oeffnen.

## Erstes Formular

1. Neues Formular erstellen.
2. Basisdaten pflegen (Titel, ID, Trigger).
3. Felder konfigurieren.
4. Sicherheit setzen (Honeypot / Captcha).
5. Versand einrichten (Mail oder Endpoint).
6. Speichern.

## Trigger nutzen

- Trigger-Link-Format: #restatify-form-{id}
- Beispiel: <a href="#restatify-form-kontakt">Kontakt</a>
- Alternativ Gutenberg-Block "Form Trigger (Restatify)" verwenden.
