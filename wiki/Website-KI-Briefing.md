# Website-KI-Briefing

## Produkt in einem Satz

Restatify Forms ist ein WordPress-Plugin fuer mehrstufige, conversion-orientierte Popup-Formulare mit starker Spam-Abwehr und flexiblem Versand per E-Mail oder API.

## Kernnutzen

- Mehr Anfragen mit fokussierten Popup-Formularen
- Schnell einrichtbar ueber einen 4-Schritt-Wizard
- Sicherheitsoptionen pro Formular (Honeypot, reCAPTCHA v3, Turnstile)
- Flexible Zustellung: wp_mail oder eigener Endpoint
- Geeignet fuer Agenturen und Websites mit mehreren Kontakt-/Lead-Flows

## Zielgruppen

- Kleine und mittlere Unternehmen mit WordPress
- Agenturen, die fuer Kunden mehrere Form-Flows pflegen
- Teams, die Leads an externe Systeme via API weitergeben wollen

## Features fuer die Website (klar kommunizieren)

1. Multi-Form-Management mit individueller Trigger-ID
2. Wizard: Basisdaten -> Felder -> Sicherheit -> Versand
3. Drag-and-drop Field Builder + Reihenfolge per Up/Down
4. Valide E-Mail/Tel-Pruefung pro Feld
5. Doppelter Mail-Flow: Owner-Benachrichtigung + Bestaetigung an Absender
6. Endpoint-Modus mit Auth (None, Bearer, Basic)
7. Gutenberg-Block fuer Trigger
